# note-taking-app - Microservices Notes App

This project is a **microservices-based note-taking application** that I built while learning backend architecture with Node.js and TypeScript.

The goal of this project is to understand how real-world backend systems are structured using **microservices, API Gateway, authentication, and multiple databases**.

This project is based on a tutorial, but I implemented and ran the services locally to learn how the architecture works.

---

# What I Learned

Through this project I practiced:

- Microservices architecture
- API Gateway pattern
- JWT authentication
- Service-to-service communication
- PostgreSQL database management
- Prisma ORM
- Docker & Docker Compose
- Structuring large backend projects

---

# Architecture

The system contains multiple services:

- API Gateway
- Auth Service
- User Service
- Notes Service
- Tags Service
